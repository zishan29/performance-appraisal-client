Guidelines
  • The faculty performance appraisal has been designed with the key deliverables of faculty members in mind. This appraisal form has five categories, which are as follows:
    1. Academic Involvement
    2. Student Development
    3. Administrative Bucket
    4. Research Bucket
    5. Consultancy and Corporate Training Bucket
  • There are some key elements of this performance appraisal form that you should be aware of before filling it out. These components are as follows:
    1. Performance Category - The main category of performance
    2. Performance Parameters - Sub category under the main category
    3. Performance criteria with quality benchmarks - Standards of perfomance
  • Quality benchmarks and weightages have been defied for almost all performance parameters. For a few performance parameters, however, each criterion is directly mapped with absolute marks.
  • Unless otherwise specified, each faculty member is required to fill out data relevant to the period of performance appraisal
  • Faculty members are required to complete the 'Faculty Information' sheet with their information.
  • The summary page is linked to entire form, and once the faculty has completed the entire form, the data will be refelected on this site. The summary sheet allows faculty to view the total score for each category.
  • Please keep the following in mind while filling the form:
    1. Faculty must respond to the question posed at the start of each performance parameter, i.e., ‘Have you performed under this parameter?‘ If the faculty member has not performed for that parameter, the faculty member must select ‘No‘ from the drop down menu. If you choose ‘No‘ the score for that parameter will be calculated ‘Zero‘ and you will not need to go through the worksheet for that parameter.
    2. If you have performed more than the specified provision (such as the number of courses/labs taught, activities organised, etc.) under a specified performance parameter, then you are required to explicitly mention the same in the tab ‘Additional Work‘
  • For calculating score for each performance parameter, following steps are to be followed
    1. Step 1: Go to the relevant performance parameter page (for example AI-1/AI-2/AI-3 etc.)
    2. Step 2: Observe the criteria for the performance parameter
    3. Step 3: Once you fill all the applicable parameters correctly, self appraisal score will be calculated for that performance
  • Marks claimed for one performance parameter cannot be claimed again for other performance parameter. Links for the relevent supporting documents (evidence) are required to be uploaded in the ‘Upload Evidence‘ section for being eligible to claim marks for that parameter.
  • If faculty memeber teaches the same subject to more than one division, only one claim for course content learning resources.
  • Faculty Who only conduct lab/tutorial sessions are required to consider the respective theory course result for p oint no. SD-2.
